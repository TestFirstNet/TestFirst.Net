Welcome to TestFirst.Net for C#.

TestFirst.Net is a library to help in creating understandable, communicative tests. TestFirst.Net is not tied to any testing library or tool. It can be used in NUnit, xUnit, Moq etc.. or your own.

The main goal of this library is to encourage writing of expressive, understandable, clear tests which new members to your project (or yourself months later) can understand as quickly as possible.
